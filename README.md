# ☕ java-basics

Exercices progressifs pour maîtriser les fondamentaux de Java - des variables jusqu'aux tableaux.

Réalisés dans le cadre de mon apprentissage de Java en parallèle de mon alternance en tant que développeur Full-Stack (React / NestJS / TypeScript).

---

## 📚 Contenu

### 🔹 Les bases — variables, opérateurs, conditions, boucles

| # | Exercice | Concepts |
|---|----------|----------|
| 01 | [Fiche identité](./Identity.java) | `String`, `int`, `double`, `boolean`, affichage console |
| 02 | [Calcul de prix](./Price.java) | Opérateurs arithmétiques, TVA, calcul TTC |
| 03 | [Conversion de température](./Temp.java) | Formule F = C * 9/5 + 32 |
| 04 | [Majeur ou Mineur](./Status) | `if / else` |
| 05 | [Calculatrice simple](./Calcul) | `switch`, `char` |
| 06 | [Note en lettre](./Note) | `switch` avec plages de valeurs |
| 07 | [Compte à rebours](./Time) | Boucle `for` |
| 08 | [FizzBuzz](./FizzBuzz) | Modulo `%`, conditions multiples |
| 09 | [Pair ou impair](./Pair) | Méthode booléenne, `%` |
| 10 | [Tableau de jeux vidéo](./Table) | `String[]`, index, boucle |
| 11 | [Calcul sur un tableau](./CalcTable) | `int[]`, somme, moyenne |

## 🛠 Stack

- **Langage** : Java 21
- **IDE** : IntelliJ IDEA / VS Code
- **Build** : aucun (exercices standalone, exécutables via `main`)

---

## 🚀 Lancer un exercice

```bash
# Compiler
javac Identity.java

# Exécuter
java Identity
```

Ou directement via le bouton **Run** dans VS Code / IntelliJ.
